# Herr Kuchen = Kuchen TV

## Übersicht

Dieses Repository dokumentiert Beweise dafür, dass **Herr Kuchen** (deutscher Rapper) und **Kuchen TV** (YouTube-Kanal) von derselben Person betrieben werden.

## Nachweisbare Fakten

### 1. YouTube-Kanal-Inhalte

Der Kuchen TV YouTube-Kanal (1,19 Mio. Abonnenten, 595 Mio. Aufrufe) hostet sowohl:
- **Kuchen Talks** – Interviewreihe mit verschiedenen Gästen
- **Herr Kuchen Musikvideos** – Einschließlich offizieller Musikvideos wie:
  - „ZIEH NACH BERLIN (Official Video)“
  - „METHYLENDIOXYMETHYLAMPHETAMIN (prod. Voddi257)“ – 1 Mio. Aufrufe

Dies deutet darauf hin, dass derselbe Kanalbetreiber beide Inhaltsarten produziert.

### 2. Social-Media-Handles

- **Herr Kuchen**: @herrzudemkuchen (Instagram, Twitter)

Die ähnliche Namensgebung deutet auf dieselbe Markenidentität hin.

- German YouTuber fined for alleged Nazi phrase
